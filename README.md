## Getting Started

### View Your Website

Your website is live at: `https://sajalex.github.io/website`

### Customize the Content

1. **Homepage** - Edit `index.html` to change the main content
2. **Styling** - Modify `styles.css` to customize the appearance
3. **Configuration** - Update `_config.yml` for Jekyll settings

### Making Changes

1. Edit the files in your preferred code editor
2. Commit and push your changes to the `main` branch
3. GitHub Pages will automatically rebuild and deploy your site (takes 1-2 minutes)
